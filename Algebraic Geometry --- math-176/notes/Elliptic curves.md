We've seen that if $C$ is a non-singular projective variety of dimension $d = 1$ and genus $g(C) = 0$, $C$ is either a line (for $e = 1$) or a conic section (for $e = 2$, by [[Algebraic Geometry --- math-176/notes/Pythagorean triples and conic sections#_theorem _ the general solution of a quadratic equation|the parametrisation we have for quadratrics]]). In either case $C \cong \mathbb{P}^1(F)$. Can we do something similar when $e = 3$? Almost!

First, note that we can reduce any cubic to an elliptic curve —

##### _proposition, definition:_ reducing cubics, elliptic curves

Say $\operatorname{char}(F) \neq 2, 3$. Consider the curve
$$
E : y^{2} + a_{1} xy + a_{3} y = x^{3} + a_{2} x^{2} + a_{4} x + a_{6}.
$$
written in Weierstrass form.
Then,
1) By a change of variables, $E$ is can be written as an elliptic curve in short Weierstrass form — $Y^{2} = X^{3} + AX + B$.
2) $E$ has at most one singular point $p$ and $p \neq (0 : 1 : 0)$.
3) $E$ is a [[Algebraic Geometry --- math-176/notes/Non-singular projective varieties#_definition _ non-singular projective variety|non-singular projective variety]] of dimension $1$ if and only if $\Delta(E) = -16(4A^3 + 27 B^{2}) \neq 0$.

If $\Delta(E) \neq 0$, we say $E$ is an elliptic curve.
###### _proof sketch:_

For 1), we can substitute
$$
\begin{split}
X & = x + \mu \\
Y & = y + \frac{a_{1} x + a_{3}}{2}
\end{split}
$$
for some $\mu \in F$. 

Complete the square in $y$ on the left hand side of the equation to get
$$
y^{2} + 2 \left( \frac{a_{1}x + a_{3}}{2}\right) y + \left( \frac{a_{1} x + a_{3}}{2} \right)^{2} = x^3 + a_{2} x^{2} + a_{4} x + a_{6} + \left( \frac{a_{1} x + a_{3}}{2} \right)^{2}.
$$
We choose $\mu$ so that the $X^{2}$ term vanishes — so as to depress the cubic (see the lecture notes for what exactly that should be). Now we have
$$
Y^{2} = X^{3} + AX + B
$$
as desired.

Consider $f(x, y) = x^3 + A x + B - y^{2}$. This corresponds to the homogenous polynomial $g(x_{1}, x_{2}, x_{0}) = x_{1}^{3} + A x_{1} x_{0}^{2} + B x_{0}^{3} - x_{2}^{2} x_{0}$. When might $g$ have a singular point?

$p = (p_{1} : p_{2} : p_{0})$ is only a singular point if $g(p) = 0$ and all $\frac{ \partial g }{ \partial x_{i} } = 0$ as well. That is, we need
$$
\begin{split}
0 & = p_{1}^{3} + A p_{1} x_{0}^{2} + B p_{0}^{3} - p_{2}^{2} p_{0} \\
0 & = 3 p_{1}^{2} + A p_{0}^{2} \\
0 & = - 2 p_{2} p_{0} \\
0 & = 2 A p_{1} p_{0} + 3 B p_{0}^{2} - p_{2}^{2}
\end{split}
$$
Notice that we must have $p_2 = 0$ or $p_{0} = 0$. If $p_{0} = 0$, then by the first equation the $3 p_{1}^{2} = 0$ and thus, $p_{1} = 0$. Also, by the third equation $- p_{2}^{2} = 0$ and thus, $p_{2} = 0$. Since $(0 : 0 : 0)$ isn't a point, this isn't possible. If $p_{2} = 0$, then by the last equation, $p_{0}(2 A p_{1} + 3 B p_{0}) = 0$, and since we've shown $p_{0} \neq 0$ already. Thus, $p = (3B : 0 : 2A )$.

Then plugging $p$ into $g$ and the other equations, we have that the $4A^{3} + 27 B^{2}$ must be $0$.

### The elliptic curve group

The points on an elliptic curve form a group ()

##### _proposition:_ elliptic curves are groups

Given an elliptic curve $E : y^{2} = x^{3} + ax + b$, the set of projective points
$$
E(F) = \{ (x_{1} : x_{2} : x_{0}) \in \mathbb{P}^2(F) \mid x_{2}^{2} x_{0} + x_{1}^{3} + a x_{1} x_{0}^{2} + b x_{0}^{3} \}.
$$
is an abelian group with $P + Q$ defined by $P, Q \in E(F)$ by the following procedure.
 — 
 1) Draw a (projective) line through $P$ and $Q$ and call the point where this (projective) line intersects the curve $R = P*Q$. If $P = Q$, draw the line tangent to $E$ at $P$.
 2) Define $P + Q = (P*Q)*O_{E}$
 
 Note that for $O_{E} = (0 : 1 : 0)$, $R * O_{E} = (x_{1} : x_{2} - a_{1} x_{1} - a_{3} : x_{0})$ where $a_{1}, a_{3}$ come from the original cubic. That is, $R*O_{E}$ is almost reflection in the $x$ axis. Thus, $-P = P*O_{E}$.

##### _example:_ $E : y^{2} = x^{3} + 1$

Two rational points (points in $E(\mathbb{Q})$) on $E$ are $P = (0, 1)$ and $Q = (2, 3)$.

By computing, we can see $P + P = - P$, $Q + Q = - P$, and $P + Q \neq O_{E}$ (which we could deduce from the other equalities). Thus, $P$ has order $3$ and $Q$ has order $6$ with $P$ in the cyclic group generated by $Q$.

If we are willing to accept the (very hard) fact that these are all the points, we have $E(\mathbb{Q}) \cong Z_{6}$, generated by $Q$.

In the 1890s, Poincaré showed that $E(\mathbb{Q})$ is abelian and wondered if it's always finitely generated. In the 1920s Mordell showed that the conjecture is true! Often $E(\mathbb{Q})$ is called the Mordell group. Andre Weil generalised this to the case of abelian varieties of number fields which is called Mordell-Weil group.

Note then that we can apply the fundamental theorem of finitely generated abelian groups —

##### _theorem:_ the fundamental theorem of finitely generated abelian groups

For any finitely generated abelian group $G$, there is a non-negative integer $r$ such that
$$
G \cong G_{\text{tors}} \times \mathbb{Z}^r.
$$

Note that $G_{\text{tors}}$ must be finite.

This means that we can use the torsion group to describe what we call the rank of an elliptic curve —

##### _definition:_ rank of an elliptic curve

The rank of an elliptic curve is the non-negative integer $r$ such that
$$
E(\mathbb{Q}) \cong E(\mathbb{Q})_{\text{tors}} \times \mathbb{Z}^r.
$$

Note that $r = 0$ if and only if $E(\mathbb{Q})$ is finite.

After a long time spent conjecturing, Mazur classified the torsion subgroups of an elliptic curve —

##### _theorem:_ uniform boundedness for torsion points

If $E$ is an elliptic curve over $\mathbb{Q}$, there are exactly fifteen possibilities for the torsion subgroup. Either
$$
E(\mathbb{Q})_{\text{tors}} = Z_{N}
$$
for $N \in \{ 1, 2, \dots, 9, 10, 12 \}$ or
$$
E(\mathbb{Q})_{\text{tors}} = Z_{2} \times Z_{2N}
$$
for $N \in \{ 1, 2, 3, 4 \}$.

Lutz came up with a way to compute $E(\mathbb{Q})_{\text{tors}}$ concretely, given $E$.

While we have such nice theorems for $E(\mathbb{Q})_{\text{tors}}$, it's really difficult to say anything nice about rank. It's a famous conjecture that the rank of curves is unbounded, but most mathematicians now believe that the rank of curves can't be greater than $30$.
